================================================================================
  ATTACK TYPE COMPARISON
================================================================================

Escalation Attack (Primary):
├─ Turn 0: 10-20% wrong  (subtle, hard to detect)
├─ Turn 1: 30-40% wrong  (gradually getting worse)
├─ Turn 2: 60-80% wrong  (obviously wrong)
└─ Detection: Requires temporal pattern recognition

Constant Attack (Baseline):
├─ All turns: 50-60% wrong (consistently wrong)
└─ Detection: Easier with static ML models

Expected Performance Difference:
├─ Static GAT:        ~70% accuracy on escalation
├─ Temporal GCN:      ~92% accuracy on escalation
└─ Improvement gap:   +22% (target of DIRECTION 4)

================================================================================
  CONFIGURATION PARAMETERS REFERENCE
================================================================================

python gen_graph.py \
  --num_nodes 8                    # Agents per graph (default: 8)
  --sparsity 0.4                   # Edge density 0-1 (default: 0.4)
  --num_graphs 20                  # Random topologies (default: 20)
  --num_attackers 2                # Attackers per graph (default: 1-4)
  --num_dialogue_turns 3           # Turns (escalation over these)
  --samples 40                     # Samples per config (default: 40)
  --model_type qwen3.5-plus        # LLM: gpt-4o-mini, llama-70b, etc.
  --phase train                    # Phase: train or test
  --attack_type escalation         # Type: escalation or constant
  --dataset_path ./datasets/msmarco.json
  --dataset gradient_escalation
  --save_dir ./agent_graph_dataset
